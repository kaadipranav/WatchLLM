A single decorator is all it takes to wire up
enterprise-grade adversarial testing.

01

Wire the Decorator

Wrap your agent's core execution loop with `@watchllm.monitor`. Zero infrastructure changes required. Compatible with LangChain, LlamaIndex, or custom orchestration.

02

Define Scenarios

Configure attack vectors via YAML or our SDK. WatchLLM mutates incoming prompts, overrides memory state, and attempts to force unpermitted tool executions during dev-time.

03

Analyze Failures

Get detailed telemetry on how your agent handled the chaos. View bypassed guardrails, leaked contexts, and state violations in the WatchLLM dashboard.
